# BoardMix 对标分析与 ViLearning 调整建议

## 1. 竞品定位速览
- **定位**：AI 驱动的无限画布协作工作台，强调「无限画布 × 无限可能」，将思维导图、流程图、PPT、看板等统一在一个产品里。
- **能力栈**：
  - **基础层**：高性能无限画布、50+ 图形组件、多媒体嵌入。
  - **AI 层**：多模态 prompt（文本/图片/网页/音频/视频/代码），一键生成导图/流程/PPT/文案。
  - **交付层**：演示模式、导入导出、权限/版本历史、模板社区。
- **使用流程**：注册→建白板/用模板→AI 生成→协作编辑→导出分享。
- **场景覆盖**：产品管理、教育培训、远程协作、创意设计。

## 2. ViLearning 可借鉴的关键点
| 维度 | BoardMix 做法 | ViLearning 可行策略 |
| --- | --- | --- |
| 画布体验 | 无限画布、所有节点都是画布元素，可自由吸附、缩放、组合 | 持续优化 Excalidraw 容器，Mindmap 节点下沉画布；引入透明 Frame、智能对齐 |
| AI 入口 | 左上角「AI 助手」汇聚导图/流程/PPT/写作等；支持多模态输入 | 近期聚焦文本输入（对接 Mindmap/Flowchart），预留图片/网页输入扩展点；AI 入口与工具栏联动 |
| 模板体系 | 模板社区提供康奈尔笔记、竞品分析等 | PRD 中补充模板里程碑：优先教育/产品管理模板，结合 AI 自动填充 |
| 工具栏 | 根据选区类型浮出专属工具（添加子节点、主题切换、样式） | 在白板优化方案中加入「Mindmap 模式工具集」「快捷键」；与流程图工具栏互斥切换 |
| 协作与交付 | 多人评论、演示、版本历史、导入导出 | 近期强调单人体验 + 导出；将协作列入阶段 D 之后的议题，预研接口（WebSocket、权限模型） |
| 教育价值 | 适合教师/学生做互动课件、头脑风暴 | ViLearning 作为 AI 学习工作台，可新增「AI 教案/课堂模板」「学习路径 Mindmap」等创新点 |

## 4. 首页文案优化方向
- **现状问题**：主页文案聚焦「AI 流程图」，对「无限画布」「多 Display」「教育场景」描述不足，也缺少差异化的 AI 教学/学习价值。
- **建议表达**：
  - 标题：「ViLearning · AI 空间化学习工作台」 → 强调无限画布 + AI 导图。
  - 副标题：「一键生成流程图、思维导图与课堂脚本，让知识在同一画布里生长。」
  - 功能卖点：
    1. *AI 智能创作*：文本描述即可生成导图/流程/课堂提纲；支持连续迭代。
    2. *双 Display 白板*：流程图与思维导图同屏联动，透明 Frame 管理知识版块。
    3. *模板启动器*：教育/产品/学习路径模板，一键生成并可自定义。
    4. *演示 & 导出*（规划中）：一键导出 PDF/PPT/图片，课堂演示模式即将上线。
  - 创新角度：突出「AI 教案 + 知识迭代」场景，与 BoardMix 的通用协作形成区隔。

## 5. 创新点（差异化）
1. **教育先行**：聚焦 AI 课堂、教案生成、论文流程图撰写、知识复盘场景，配合模板与学习路径 Mindmap。
2. **上下文引擎**：结合 `contexts` 表做选区上下文、教材片段的复用，形成「AI 记忆」体系。
3. **Display Registry × 模板**：支持自定义 Display + 模板驱动的 Prompt Pipeline，让用户快速扩展新的知识载体。
4. **学习闭环**：Mindmap/Flowchart 与测验、复盘笔记互通，真正实现“生成 → 学习 → 检测”的空间化体验。
