# 前言
1. 核心定位：融合 “人生规划” 与 “白板办公桌” 双场景的可视化协作工具，聚焦留学生（商科）规划、ADHD 学习者多线程学习两大核心用户群。
2. 视觉基调：统一采用赛博朋克风格，主色调为蓝色、紫色。
3. 核心逻辑：以 “一级纵向思维导图（Flowith 形式）+ 二级互动内容块（Kuse 形式）” 为架构，实现资料上传 - 可视化转换 - 拖拽编辑 - 批量生成的全流程闭环。



# 一、文档说明
## 名词解释
术语/缩略词｜说明：
CMS	内容管理系统（Content Management System），此处指代平台内容编辑后台
DAG	有向无环图（Directed Acyclic Graph），支撑工作流调度的核心数据结构
一级模块	纵向思维导图（Flowith 形式），为内容聚合的顶层框架，支持批量生成
二级内容	白板上的单个聊天记录及 Display 结果（Kuse 形式），挂载于一级模块下
Display	资料转换后的可视化形态，含横向思维导图 / 笔记 / 卡片 / 题目四种选择（ThetaWave 形式）
ThetaWave	参考 ThetaWave 工具的 Notebook LLM 资料形式及可视化展示风格
Cursor/Boardmix	参考 Cursor、Boardmix 的右侧对话框交互逻辑，用于触发资料转换操作
Flowith	参考 Flowith 的模块形态，为一级纵向思维导图的核心样式
Kuse	参考 Kuse 的内容块样式，为二级内容在白板上的呈现形态

## 需求范围
可条理性地罗列需求范围或信息架构
1. 视觉风格层：全平台统一风格，在已有风格基础上，参考/Users/miles/Downloads/code/flowchartai/docs/前端引导文件夹下的前端设计引导进行设计，主色调为蓝色、紫色，所有模块 / 内容块需匹配该视觉基调。
2. 资料上传与转换层：支持 ThetaWave 形式的 Notebook LLM 资料上传，通过右侧 Cursor/Boardmix 风格对话框触发转换，输出横向思维导图 / 笔记 / 卡片 / 题目四种 Display 形态（ThetaWave 形式）。
3. 白板内容管理层：支持 “添加” 操作，将单个聊天记录 + Display 结果以 Kuse 形式挂载至白板；核心架构为 “一级纵向思维导图（Flowith 形式）+ 二级内容块”，二级内容需挂载于一级模块下。
4. 互动编辑层：二级内容块支持文字编辑、节点拖拽（同思维导图内 / 跨思维导图）、节点连线；一级纵向思维导图模块支持批量生成。
5. 垂直场景层：覆盖两大核心场景 ——① 人生规划（商科留学生留学 / 求职规划）；② 白板办公桌（ADHD 学习者多线程学习、能动性提升）。
